# 再生塑料品名体系 · 基础文件 v1.0

用途：再生工厂供应发布 + 制品/改性/色母工厂采购需求的品类标准字典
数据来源：91再生、变宝网、再塑宝、绿塑黄埔四大平台+多家再生颗粒厂商产品目录
编制：国嘉基业·LLM Wiki | 2026-06-28
原则：纯再生料（颗粒/切片/改性料），不含任何废塑料（破碎料/瓶砖/瓶片/压包料）

## 与废塑料体系的边界

| 再生料（本文件） | 废塑料（另见废塑料品名体系文件） |
|---|---|
| 再生颗粒（熔融造粒） | 瓶砖/压包料 |
| 改性颗粒（添加助剂/玻纤） | 破碎料/粉碎料 |
| 再生切片（PET瓶级切片） | 毛片 |
| 泡泡料（团粒） | 水口料/胶头 |
| 再生粉（研磨粉） | 未退漆片料 |
| 副牌颗粒 | 未洗任何形态 |
