# 废塑料品名体系 · 基础文件 v1.0

用途：打包站发布供应 + 再生工厂采购需求的品类标准字典
数据来源：91再生、变宝网、再塑宝、绿塑黄埔四大平台全网采集
编制：国嘉基业·LLM Wiki | 2026-06-28
原则：纯废塑料，不含任何再生料（无颗粒、无洗片、无造粒料）

## 硬规则

| 废塑料（本文件范围） | 排除项（再生料范围） |
|---|---|
| 瓶砖/压包（干法分拣压包） | 任何带"洗"字（热洗片、冷水片、水洗料） |
| 破碎料/粉碎料（干法机械破碎） | 任何"颗粒/造粒料" |
| 水口料/胶头/机头料（工业边角） | PET"片"（已清洗） |
| 退漆料（预处理，未熔融） | "净片"（水洗过） |
| EPS热熔块/冷压块（物理压缩） | "再生料/再生颗粒" |
| "净砖"（分拣去标，未水洗） | 泡泡料 |
| 毛片（未洗粉碎料） | |

### 形态分布

| 形态 | 说明 |
|---|---|
| 压包料 | 收集后直接压缩打包，未经破碎 |
| 破碎料/粉碎料 | 经干法机械破碎/粉碎，未经水洗 |
| 毛片 | 干法破碎但未经清洗的片状料 |
| 水口料/胶头 | 注塑工厂产生的浇口、流道、清机料 |
| 块料 | EPS热熔/冷压块、PC块料等 |
| 散料 | 未压包的散瓶、散膜等 |
