## 二、采集引擎

文件：scrapers/scraper.py

- Python 3.12，requests + BeautifulSoup
- 模块化解析器架构（每个平台独立 Parser）
- 自动标准化输出格式（platform/type/category/material/title/quantity/price/location/company/contact/date/url）
- 错误容错：超时/404/403/429 自动跳过，重试2次
- 反反爬：随机User-Agent、请求间隔1.5-3.5秒
- 增量保存：每日JSON + JSONL累积
运行方式：

cd scrapers/
./run_daily.sh              # 全量采集
python3 scraper.py          # 同上
python3 scraper.py --platform zz91 --max-pages 5  # 指定平台
