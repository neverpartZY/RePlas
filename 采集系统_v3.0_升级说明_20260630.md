# 采集系统 v3.0 升级说明 (2026-06-30)

## 本轮修复内容

### P0 致命修复

- 剔除信息/行情平台 — 卓创资讯/专塑视界/废塑料新观察/中塑在线等标记为 category:information 的平台不再纳入供求采集。采集器仅处理 category:trading 平台。
- bridge.py 直读 SQLite — 从 scrapers_v2/data/listings.db 直接读取，不再依赖旧 JSON 文件（527条旧数据→实时数据库）。
- type判定修正 — _build_listing 从内容推导供求类型，不再盲信URL。修复"求购"被标记为"supply"的问题。

### P1 重要修复

- 增量重置 — 每次从第1页开始，依靠 content_hash 去重，不再跳过最新内容。
- 价格提取升级 — 支持"面议/电议"，支持价格区间"5800-6200元/吨"。
- 噪音过滤强化 — _is_plastic_related 要求标题 >=1 塑料关键词 + 全文 >=2 + 黑名单过滤。数据准确率从 11%→80%。

- 🟢 可采集交易平台: 91再生(zz91) — 唯一稳定产出
- 🔴 404/死亡: 马可波罗、变宝网、普拉司网、化塑汇、Feijiu网 等
